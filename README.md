# windows environment validation

c++ program that runs validation checks to determine whether execution is occurring on a genuine microsoft windows system.

## usage
compile from source yourself or download prebuilt binaries from [https://github.com/lain804/winedetect/releases/tag/Release](https://github.com/lain804/winedetect/releases/tag/Release)

## demo
screenshot comparing results between wine on vmware (ubuntu) and native windows 10
1 means a check passed, 0 means it failed
